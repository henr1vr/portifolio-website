# Portifolio-website
This is a portfolio website for Henri van Rensburg. It aims to showcase a few design projects, introduce Henri, and allow visitors to contact him.
# Section breakdown
## Header
The header acts as an introduction to the website. It explains what the website is and provides a core statement about who Henri is as a designer.
## Navigation
The navigation presents the user with links to each of the sections on the home page. Each link will scroll to its relevant section when clicked.
## My Work
This section presents UI cards for a selection of design projects. Each project card will link to the project detail page for that specific project.
## About Me
This section speaks about who Henri is, what his qualifiactions are, and shows a timeline of his experience.
## Get In Touch
This section includes a simple contact form to allow users to send an email to Henri.
# Multimedia Elements
For now, the My Work section only includes a single mockup image of one of the projects. This section will expand and include poster images in the form of UI cards. Each project page will include multiple images of UI designs.
The About Me section will include an image of Henri.
The hero section will include an embeded iframe from Spline3D for an interactive element.
# Styling
The website will have a sleek visual finish with some retro elements like pixelated fonts, a friendly robot character, and colourful UI pills. The Home page will switch between light and dark sections and make use of colourful UI elements to elevate its visual precense.
Each project page will be styled to match the design of the project being showcased.
